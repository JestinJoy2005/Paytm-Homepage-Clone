Paytm-Homepage-Clone

This is a frontend Paytm clone created using **HTML and CSS**.  
The project is designed to replicate the basic layout and user interface of the Paytm website.

It includes a homepage with a navigation bar, top banner, recharge & bill payment services section, promotional banners, and feature icons.

This project was built for learning and practice purposes to improve frontend web development skills.

Features
- Paytm-style homepage layout  
- Navigation bar with Paytm logo and menu options  
- Top promotional banner section  
- Recharge & bill payment services with icons  
- Right-side promotional image banner  
- Bottom promotional banners  
- Fully styled using **pure CSS**  

Technologies Used
- HTML  
- CSS  

Purpose
This project was created to practice:
- HTML page structure  
- CSS styling and layout  
- Real-world website UI design
